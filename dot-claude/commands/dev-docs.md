승인된 계획을 기반으로 dev docs를 생성합니다.

1. dev/active/[task-name]/ 디렉토리를 생성한다
2. 세 개의 파일을 생성한다:
   - [task-name]-plan.md: 방금 승인된 계획 전문
   - [task-name]-context.md: 관련 파일 경로, 아키텍처 결정, 의존성
   - [task-name]-tasks.md: 단계별 체크리스트 (체크박스 형식)
3. context 파일에는 'Last Updated' 타임스탬프를 포함한다
4. 추가 조사가 필요하면 컨텍스트가 남아있는 한 수행한다
